In this project, we are using a basic substitution cipher algorithm to encrypt and decrypt the plain text. The encrypt function takes the plain text and a key as input and returns the cipher text after applying the substitution cipher algorithm. The decrypt function takes the cipher text and the same key as input and returns the plain text after reversing the substitution cipher algorithm. In the main function, we have initialized a plain text string and a key value. We then call the encrypt function with these values to encrypt the plain text and store the encrypted text in a variable. We then call the decrypt function with the encrypted text and the same key to decrypt the text and store it in another variable. Finally, we print the original text, the encrypted text, and the decrypted text using the cout statement.
